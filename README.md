# IGR205_Project

Implementation of a Sketch-based 3D Modeling and Rigging interface.

## Build and run

**Build:**
From the project directory:
```
cmake src -B build
make -C build
```

**Run:**
From the projet directory:
```
./src/sketch-based-modeling
```

## Libraries used
* **Glad**
* **GLM**
* **GLFW**
* **STBImage** for png and jpeg image reading and writing.
